# System Patterns: Pico

## Architecture
- **Frontend:** React with TypeScript.
- **Build Tool:** Vite.
- **Styling:** TailwindCSS with PostCSS.
- **Package Management:** pnpm.

## Key Technical Decisions
- **Vite:** Chosen for its extremely fast HMR and build performance.
- **TailwindCSS:** Provides a utility-first CSS framework for rapid UI development.
- **TypeScript:** Ensures type safety and improves maintainability.

## Component Structure
- `src/App.tsx`: Main entry component.
- `src/main.tsx`: Application entry point.
- `src/index.css`: Global styles including Tailwind directives.
